Detailed Description

Next attempt to make a better Stabilizer. In the main phase it fixes the rotations of the objects in the world. Then it lets the physics engine execute, alternatively moving the objects only in y-direction, then only in x-direction, then repeating this. Finally it will simulate in both x-and y-direction together, and then it will turn off fixed-rotation again and simulate further. ( There is still enough room for improvement for future endeavors )
