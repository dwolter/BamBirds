Plan de.uniba.sme.bambirds.planner.PrologPlanParser.parsePlan ( String  plan) throws JsonParseException

Parses a String of a given format into a Plan Object. It operates on the format of the new Prolog code of 2021, thus it can not be used with the planner used before.

The format used is [target,impactAngle,chosenStrat,confidence,[reasonA,reasonB,...]?,shotInfo?] for one Plan, where the values with ? are optional. The first and last brace may be omitted

List< Plan > de.uniba.sme.bambirds.planner.PrologPlanParser.parsePlans ( String  plans)

Parses a Prolog plan of the new (2017) format. The Prolog-module of our agent only returns a String representation of his results, which has to be processed into Plans.

The format used by Prolog is: [Plan_0,Plan_1...,Plan_N]
