double de.uniba.sme.bambirds.common.objects.ExecutedNode.similarityTo ( Node  node)

Calculate the similarity of another node to the executed node. The score is calculated using the following metrics:

  • Impact Angle
  • Location of target object
  • Bird type
  • Expected Shot effects
Parameters
nodeAnother node
Returns
A score between 0 and 1 stating the similarity. 1 means identical, 0 means not at all similar
