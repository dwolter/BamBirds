Detailed Description

The DefaultPreprocessor serves as a good startingpoint/example-implementation for preprocessing a raw scene that was created based on a Computervision-Level. It applies a finetuned sequence of SceneModifiers which works pretty well for the first 20 Levels of the Angry-Birds-Game.
