Detailed Description

Base class for Sceneentities already preimplementing some of the required methods. Deriving classes may need to reimplement some of the methods ( e.g. scaling or rotating ) depending on the shape in order to behave correctly. A SceneEntity stores a reference to the original Computer-Vision-Body from which it also derives some of its initial static state ( id, mbrSize, position, angle...). Whenever we create a Snapshot out of a running world, some of the entities state will be dynamically updated ( position, angle, velocities). The EntityProperties define the entities physical behavior once it is instanced as a body in a world. They are derived from the ABType of the entity (e.g wood, stone, ice etc.).
