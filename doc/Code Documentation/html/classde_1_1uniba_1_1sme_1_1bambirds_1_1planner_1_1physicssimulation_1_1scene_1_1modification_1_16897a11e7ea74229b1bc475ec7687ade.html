Detailed Description

attempts to counter computer-vision-inaccuracies. The modifier will go through each polygon one by one. It will try to to fight small inaccuracies and make edges straighter. If multiple vertices are probably meant to sit on an "invisible line" in x-direction or y-direction (whithin the given maxdistance), the vertices will be shifted to the most occuring value on that "line". If there are only 2 vertices within that distance, one of them will be shifted to the others value ( by random choice ), so that they form a straight edge.
